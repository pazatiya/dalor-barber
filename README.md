# Yair Barber Booking

אפליקציית קביעת תורים עצמאית למספרה של יאיר.

## הרצה מקומית

```bash
npm install
npm start
```

האפליקציה תרוץ בכתובת:

```text
http://localhost:3020
```

## מה יש בפנים

- מסך פתיחה עם לוגו DALOR
- בחירת שירות, יום ושעה
- שם מלא, טלפון והערות
- שמירת תורים לקובץ `data/appointments.json`
- סימון שעות תפוסות
- פתיחת הודעת וואטסאפ מוכנה אחרי שמירת תור

## שינוי מספר וואטסאפ

בקובץ `public/index.html`, שנה את הערך:

```js
const WHATSAPP_NUMBER = "972507983306";
```

למספר של יאיר בפורמט בינלאומי, בלי `+`.
